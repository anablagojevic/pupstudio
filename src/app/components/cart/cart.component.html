Shipping and returns

Unless it has issues to the state of food, once opened we regret that we cannot, by law, accept returns of any Food, Treats and Supplements including all flea and tick’s treatments and wormers.

We are happy to offer a Return for any item within 7 days after receipt. The item must be re-saleable (by this, we mean that it can be sold to another customer in an ‘as new’ state.) Please make sure to put the item in a box as you received it before returning it.

Shipping cost will be deducted from your refund for the return fees.
